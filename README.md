# Grzegorz Kmita | Portfolio

Personal portfolio website showcasing my experience as a Threat Prevention Engineer, featuring work history, skills, conference talks, and education.

## 🌐 Live Site

[otisthescribe.github.io](https://otisthescribe.github.io)

## 🛠️ Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Custom properties, Grid, Flexbox, animations
- **Vanilla JavaScript** - Matrix rain effect, scroll animations, smooth navigation

## 📁 Project Structure

```
├── index.html          # Main HTML file
├── css/
│   └── styles.css      # All styles
├── js/
│   └── main.js         # JavaScript functionality
├── assets/
│   └── images/         # Image assets (if any)
├── .gitignore
├── LICENSE
└── README.md
```

## ✨ Features

- Responsive design (mobile-first approach)
- Matrix rain background animation
- Smooth scroll navigation
- Reveal-on-scroll animations
- Interactive skill tags and cards
- Terminal-style UI elements

## 🚀 Local Development

Simply open `index.html` in your browser, or use a local server:

```bash
# Python 3
python -m http.server 8000

# Node.js (with http-server installed)
npx http-server
```
